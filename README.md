# WME-BackEnd-Data
![WME-BED](https://github.com/TheCre8r/WME-BackEnd-Data/blob/master/images/WMEBED.PNG?raw=true)

## Synopsis

Adds the ability to view Hidden External Providers, view gas prices, and advertisement pins (Ad pins).
